# Introduction

claude-channels is a collection of MCP channel plugins that bridge external chat platforms (Slack, Line, etc.) with Claude Code sessions. Each plugin is an MCP server that enables two-way messaging — users send messages from their preferred chat platform, and Claude replies back through the same channel.

## How it works

Each channel plugin runs as a local MCP server subprocess alongside your Claude Code session. The plugin:

1. Connects to your chat platform (e.g., Slack via Socket Mode)
2. Creates a dedicated thread for the session
3. Forwards incoming thread messages to Claude Code as MCP notifications
4. Provides tools for Claude to reply, react, and edit messages within the thread

## Prerequisites

- [Claude Code](https://docs.anthropic.com/en/docs/claude-code) CLI installed
- A Slack workspace with admin access (for Slack plugin)

## Installation

### Option 1: Marketplace plugin (recommended)

Install the plugin from the Claude Code marketplace:

```bash
claude plugin add claude-channels/slack
```

Then configure your credentials:

```bash
/slack:configure <xoxb-bot-token> <xapp-app-token>
```

### Option 2: Development mode

For local development or testing unreleased changes:

```bash
git clone https://github.com/pleaseai/claude-channels.git
cd claude-channels
bun install
```

Then load the plugin in development mode:

```bash
claude --dangerously-load-development-channels plugin:slack@claude-channels
```

### Configure your platform

Each plugin requires platform-specific credentials. Create the state directory and add your credentials:

```bash
mkdir -p ~/.claude/channels/slack
```

Choose one of two modes:

**Channel mode** — threads in a shared channel:

```env
CLAUDE_SLACK_BOT_TOKEN=xoxb-your-bot-token
CLAUDE_SLACK_APP_TOKEN=xapp-your-app-token
CLAUDE_SLACK_CHANNEL_ID=C0123456789
```

**DM mode** — threads in a direct message with the bot:

```env
CLAUDE_SLACK_BOT_TOKEN=xoxb-your-bot-token
CLAUDE_SLACK_APP_TOKEN=xapp-your-app-token
CLAUDE_SLACK_DM_USER_ID=UYourSlackUserID
```

Get your user ID from Slack: **Profile > ⋮ > Copy member ID**

### Start a session

Launch Claude Code. The plugin creates a new thread in the configured channel (or DM) with "Claude Code session started". Send messages in that thread to interact with Claude.
