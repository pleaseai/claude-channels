# Introduction

claude-channels is a collection of MCP channel plugins that bridge external chat platforms (Slack, Line, etc.) with Claude Code sessions. Each plugin is an MCP server that enables two-way messaging — users send messages from their preferred chat platform, and Claude replies back through the same channel.

## How it works

Each channel plugin runs as a local MCP server subprocess alongside your Claude Code session. The plugin:

1. Connects to your chat platform (e.g., Slack via Socket Mode)
2. Creates a dedicated thread for the session
3. Forwards incoming thread messages to Claude Code as MCP notifications
4. Provides tools for Claude to reply, react, and edit messages within the thread

## Prerequisites

- [Bun](https://bun.sh) 1.3.10 or later
- [Claude Code](https://docs.anthropic.com/en/docs/claude-code) CLI installed
- A Slack workspace with admin access (for Slack plugin)

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/pleaseai/claude-channels.git
cd claude-channels
```

### 2. Install dependencies

```bash
bun install
```

### 3. Configure your platform

Each plugin requires platform-specific credentials. For the Slack plugin, you need a Slack app with Socket Mode enabled.

Create the state directory and add your credentials:

```bash
mkdir -p ~/.claude/channels/slack
```

Create `~/.claude/channels/slack/.env` with your Slack tokens:

```env
SLACK_BOT_TOKEN=xoxb-your-bot-token
SLACK_APP_TOKEN=xapp-your-app-token
SLACK_CHANNEL_ID=C0123456789
```

### 4. Register the MCP server

Add the channel plugin to your Claude Code MCP configuration (`~/.claude/settings.json` or `.mcp.json`). Use the absolute path to the plugin:

```json
{
  "mcpServers": {
    "slack-channel": {
      "command": "bun",
      "args": ["run", "/path/to/claude-channels/plugins/slack/server.ts"]
    }
  }
}
```

Replace `/path/to/claude-channels` with the actual path where you cloned the repository.

### 5. Start a session

Launch Claude Code. The plugin creates a new thread in the configured channel with "Claude Code session started". Send messages in that thread to interact with Claude.
